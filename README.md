# 🎓 Student Management System (Java)

A simple and efficient console-based application to manage student records using Core Java.

---

## 🚀 Features

- Add Student  
- View All Students  
- Delete Student  
- Search Student by ID  
- Update Student Details  

---

## 🛠️ Technologies Used

- Java  
- ArrayList (Data Structure)  

---

## ▶️ How to Run

1. Compile the program:
   javac Main.java

2. Run the program:
   java Main
